
       Wget Downloader for Windows XP, 2003, Vista, Win 7, 8, 10 (32 and 64 bit)

                                           Version 1.0 Released May 28, 2023
========================================================================


Description
-----------
A simple Wget Downloader application with a graphical user interface (GUI) using the Tkinter library. The application allows users to download files from a specified URL and save them to a selected output directory.
  

Installation
------------
 1. Double-click "Wget Downloader.setup" to start the installation process
 2. Follow the on-screen instructions
 3. Wait for the installation to complete
 4. Launch the software

That's it! You have successfully installed Wget Downloader on your computer. I hope this guide was helpful, and feel free to   ask if you have any questions or need further assistance.


Dependencies
------------------
The application relies on the following dependencies:

- Wget command-line tool (should be installed and accessible in the system's PATH environment variable)


Usage
--------
  1. Run the executable file (.exe) to launch the Wget Downloader application.
  2. The main window of the application will appear.
  3. Enter the URL of the file you want to download in the provided entry field.
  4. You can also use the "Paste URL" button to automatically populate the URL field with the     contents of your clipboard.
  5. Select the output directory where you want to save the downloaded file by clicking the     "Browse" button. A file dialog will open, allowing you to choose a directory.
  6. Once you have entered the URL and selected the output directory, click the "Download"     button to initiate the download process.
  7. The application will use the Wget command-line tool to download the file from the     specified URL and save it to the selected output directory.
  8. The status label will be updated to display the progress of the download process.
  9. Once the download is complete, a pop-up window will appear, indicating the successful     completion of the download.


Uninstallation
--------------
Run Uninstall from 'Wget Downloader' folder in the Start menu or go to 
Control Panel, launch "Add or Remove Program" applet and find the program here.
  
  Please note, you do not need to unistall the program before installation of a 
  new version! Just install it over an old version.
